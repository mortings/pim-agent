Engine backend (Cloudflare Worker)

This page stores only the Worker URL + shared passphrase. Every Bluestone credential lives as a Worker secret, so it never reaches the browser.

Reads & write-backs to Bluestone use the Worker's MAPI/PAPI credentials — set once, server-side, never entered here:
wrangler secret put BLUESTONE_PAPI_KEY wrangler secret put BLUESTONE_MAPI_CLIENT_ID wrangler secret put BLUESTONE_MAPI_CLIENT_SECRET wrangler secret put WEBHOOK_SECRET
